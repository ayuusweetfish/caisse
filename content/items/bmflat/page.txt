{{contents.en=}}
<b Bmflat/Bm♭> is a program library in the C language for parsing BMS (Be-Music Source). BMS is a musical track format commonly used by rhythm games. This library can be useful when implementing games or analysis tools.
!<hr>
Two programs have been crafted to demonstrate its possible use cases. Flatspin, included in the source package, is an interactive playback and visualisation tool for BMS music tracks. <link https://gist.github.com/50764b32880710f9ec8b95de353a18fb BGA Compo> is a utility that renders audio and background animation videos (BGAs) from BMS files, which may be useful for track previews — many tracks from the old times do not include complete video files and instead employs two layers of fast-changing still images superimposed to form the animation; but these retro-style technical limitations seem to be inspiring a lot of creativity!
